## 1. Project Description

- **MyBike**: A landing page about bicycle sales. It's a demo site that shows users the availability of specific bicycles and their advantages. The site has sections such as:
  - Home
  - About Us
  - Compare Bikes
  - Details
  - Contacts

  each of which is included in the hamburger menu. Clicking on each section smoothly scrolls the user to the section's content, where they can see what each section is about. The site is also responsive, making it convenient for mobile use.
